## Data Source

This project uses public 311 Service Requests data from the City of Chicago Data Portal.

Dataset:
Chicago 311 Service Requests  
https://data.cityofchicago.org/Service-Requests/311-Service-Requests/v6vf-nfxy

Timeframe used in analysis:
January 2024 â€“ September 2024

Due to GitHub file size limits, raw CSV files are not stored in this repository.

### How to reproduce the data
Data can be downloaded directly from the City of Chicago portal by filtering
CREATED_DATE between 2024-01-01 and 2024-09-30 and exporting as CSV.
